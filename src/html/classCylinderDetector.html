Detailed Description

CylinderDetector class that handles detection of 30cm diameter cylinders from laser scan data and draws them on a occupancy grid map using OpenCV for image processing. Needs to have a base map published to the topic "/map" to draw on and publishes the modified map to the topic "modified_map".
