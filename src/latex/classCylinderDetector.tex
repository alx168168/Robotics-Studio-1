\doxysubsection{Detailed Description}
\mbox{\hyperlink{classCylinderDetector}{Cylinder\+Detector}} class that handles detection of 30cm diameter cylinders from laser scan data and draws them on a occupancy grid map using Open\+CV for image processing. Needs to have a base map published to the topic \char`\"{}/map\char`\"{} to draw on and publishes the modified map to the topic \char`\"{}modified\+\_\+map\char`\"{}.
